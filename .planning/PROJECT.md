# 修仙欠费中（暂定名）

## What This Is

这是一款面向喜爱修仙题材、希望体验高压社会模拟的玩家的 **Web 生存策略游戏**（Nuxt/Vue 单机页游）。玩家扮演背负债务的修仙高中生，在“分数=权限”的制度中分配每日有限时间，在修炼、上课、打工、休息与补给之间做残酷取舍。游戏的重点不是传统 Game Over，而是通过持续的小崩溃与结构性压迫，让玩家感受到被系统逐步驯化直至麻木的过程；整体偏生存沙盒，支持无限天数推进并在过程中触发情节结局。**v1.0 已交付**：核心循环、制度压力、事件管线、心理主题表达、本地存档与响应式主流程均已落地。

## Core Value

让玩家在完整一局后切身感受到：在某些系统面前，“努力”也可能只是更高级的自我奴役。

- ✓ 玩家可以在无限天数循环中进行“修炼、上课刷分、打工还债、休息、买补给”行动决策，并形成资源闭环 — **v1.0**
- ✓ 玩家可以持续经历“上课刷分 vs 打工还债”的核心冲突，并在分班、资源与债务压力中产生可感知的长期后果 — **v1.0**

- ✓ 玩家可以使用本地存档进行纯单机体验（无账号、无联网、无社交） — **v1.0**

## Context

- 世界观与情绪目标：修仙题材包裹下的高压制度生存模拟，强调结构性压迫与“被驯化”体验
- 目标用户：喜欢策略生存与叙事压力体验、愿意反复重开的核心玩家
- 局制设计：无限天数推进，过程中可触发情节结局；以周考节律形成长期压迫曲线
- 关键系统关系：时间段稀缺驱动每日取舍；分数决定权限与资源；债务驱动现金压力；补给带来短期收益与长期身体劣化
- 产品边界：**v1.0 已达成** 单机核心循环与主题表达；扩展能力留待下一里程碑

## Constraints

- **Platform**: Web 单机优先，需兼容桌面端与移动端响应式布局
- **Scope**: v1 聚焦核心循环与情绪表达；后续里程碑单独控范围
- **Data**: 本地存档且无账号依赖（v1）；未来若上云需单独里程碑
- **Session**: 双模式偏沙盒（无限天推进 + 情节结局触发）
- **Narrative Tone**: 保持“无绝对死亡、但持续精神驯化”的表达一致性

## Key Decisions

| Decision | Rationale | Outcome |
|----------|-----------|---------|
| v1 采用双模式（无限天 + 过程中情节结局） | 偏生存沙盒，避免固定天数硬切断；结局由状态与事件触发 | ✓ v1.0 已验证 |
| v1 采用纯单机+本地存档 | 聚焦私密情绪体验，降低复杂度 | ✓ v1.0 已验证 |

| 失败定义采用“精神驯化”而非硬性 Game Over | 强化辨识度与主题冲击 | ✓ v1.0 已验证 |
