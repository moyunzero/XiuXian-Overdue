# 修仙欠费中 - 项目文档

## 项目概述

**修仙欠费中** (xiuxian-qianfei) 是一款以修仙世界观为外壳的高压社会生存模拟游戏。玩家扮演背负沉重债务的修仙高中生，在"分数=权限"的制度下挣扎求存。

## 技术栈

| 类别 | 技术 |
|------|------|
| 框架 | Nuxt 4.3.1 + Vue 3.5.30 |
| 语言 | TypeScript |
| 3D 图形 | Three.js 0.183.2 |
| 测试 | Vitest 4.1.0 |
| 样式 | 原生 CSS (CSS Variables) |
| 字体 | IBM Plex Mono + ZCOOL XiaoWei |

## 核心功能

### 游戏引擎
- 纯函数实现，无副作用
- 确定性计算，便于测试
- 支持存档持久化

### 经济系统
- 债务本金 + 利息 + 系统费用池
- 每周复利机制
- 分班制度影响利率

### 心理系统
- 驯化值 / 麻木值
- 请神契约系统
- 崩溃事件与叙事延迟

### 3D 可视化
- Three.js 人体模型
- 身体部位偿还可视化

- 优先主线：方案 A「修仙信用社会模拟器」
